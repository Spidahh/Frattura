# FRATTURA - START

La cartella ora e' organizzata in modo semplice.

Apri prima:

- `00_COMANDO/00_START.md`

Struttura:

- `00_COMANDO/` = cabina di comando viva.
- `01_MATERIALE_DA_RECUPERARE/` = tutto il materiale narrativo vecchio, letto e da recuperare/tagliare/trasformare.
- `02_PRODUZIONE_TECNICA/` = script, PDF, output tecnici.
I vecchi indici ridondanti sono stati spostati in `00_COMANDO/_dettagli_revisione/indici_vecchi_pre_simplificazione/`.

Regola:
non si parte piu' da `story_bible`, `bozze` o `_archivio`.
Si parte da `00_COMANDO/00_START.md`.
