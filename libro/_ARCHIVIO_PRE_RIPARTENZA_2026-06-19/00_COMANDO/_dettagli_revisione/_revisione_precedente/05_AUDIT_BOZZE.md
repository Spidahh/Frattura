# 05 - AUDIT BOZZE

Questo file auditera' le bozze del Libro 1.

Regola:
- non si giudica se un capitolo "suona bene";
- si giudica che cosa si puo' recuperare per la nuova storia.

---

## Criteri per ogni capitolo

Ogni capitolo va classificato in cinque colonne:

1. `Recuperare`
   - luogo, immagine, dialogo, azione, atmosfera, rapporto.

2. `Tagliare`
   - Marchio, braccio, potere, prescelto, porta, compatibilita', spiegoni vecchi.

3. `Trasformare`
   - scena valida ma funzione vecchia.

4. `Nuova possibile funzione`
   - cosa potrebbe fare in una nuova architettura.

5. `Priorita'`
   - alta / media / bassa.
