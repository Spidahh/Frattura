# 00 - START / CABINA DI COMANDO

Questo e' l'unico file da aprire per primo.

## Stato

Il progetto Frattura e' in rifondazione.
La vecchia trama e' bocciata.
Il materiale vecchio non viene buttato: viene recuperato, tagliato o trasformato.

| Cartella | Uso |

| `01_MATERIALE_DA_RECUPERARE/` | vecchio canone, bozze, grafica, archivio storico |
| `02_PRODUZIONE_TECNICA/` | script e PDF, non narrativa |

## Regola assoluta

Nessuna nuova trama prima di aver controllato:

- vincoli autore;
- materiale recuperabile;
- cose vietate;
- decisioni gia' prese.

## Comando rapido

Se devi lavorare al libro:

1. controlla `01_REGOLE_STORIA.md`;
2. guarda l'inventario in `03_INVENTARIO_RECUPERO.md`;
3. aggiorna `04_DECISIONI_E_BACKLOG.md`;
4. controlla lo schema logico in `05_SCHEMA_LOGICO_DIREZIONE.md`;
5. controlla il recupero totale in `06_RECUPERO_TOTALE_MATERIALE.md`;
6. controlla finale provvisorio e matrice in `07_FINALE_PROVVISORIO.md` e `08_MATRICE_MISTERI.md`;
7. controlla i motori in `09_TRE_MOTORI_SAGA.md`;
8. solo dopo produci audit, nuova premessa o struttura.
