# 04 - DECISIONI E BACKLOG

## Decisioni approvate

| Decisione | Impatto |
|---|---|
| Rifare trama e canone | la vecchia direzione non comanda piu' |
| Kaelen non speciale | via Marchio speciale su Kaelen, braccio/porta/compatibilita' |
| Mystery prima di tutto | no horror macabro come impronta |
| Fantasy strutturale | mondo e regole devono contare |
| Vaekh e' solo la citta' iniziale | il mondo oltre Vaekh va recuperato e sviluppato |
| Saga piena di colpi di scena | nulla deve restare esattamente come sembra, se e' importante |
| Personaggi ricorrenti | anche figure marginali possono tornare e diventare fondamentali |
| Finale provvisorio obbligatorio | non si costruisce la saga senza sapere dove puo' chiudere |
| Archivio letto e indicizzato | non si butta materiale senza inventario |
| Cartella semplificata | root pulita, comando unico |
| Recupero totale in file vivo | materiale strutturale salvato in `06_RECUPERO_TOTALE_MATERIALE.md` |
| Finale provvisorio creato | base tecnica in `07_FINALE_PROVVISORIO.md`, da approvare o demolire |
| Matrice misteri creata | base tecnica in `08_MATRICE_MISTERI.md`, da ampliare |
| Tre motori di saga creati | alternative strutturate in `09_TRE_MOTORI_SAGA.md` |
| Marchio non vietato in assoluto | ammesso solo se rifondato, non su mano/polso/braccio, non fondativo, non da prescelto |

## Decisioni bocciate

| Bocciato | Non ripetere come |
|---|---|
| Marchio banale/centrale | segno su mano/polso/braccio, potere personale, protagonista eletto, spiegazione della saga |
| Prigioniero vecchio | entita' finale, spiegone cosmologico, Accordatore finale |
| Veglia vecchia | rito/protocollo pronto come perno |
| Dormiente sotto la citta | dio, mostro, regno, prigioniero sotto Vaekh |
| Passato segreto unico | "era tutto diverso" come solo twist |
| Macabro dominante | organi, tributi, carne, citta-corpo |
| Racconto pubblico/versione dei fatti | storia vivibile, consenso, guerra del racconto, Mira che vince il racconto |
| Vaekh come mondo totale | dimenticare Primo Mondo, esterni, geografia e storia larga |
| Personaggi usa-e-getta | comparse che non possono tornare, secondari senza traiettoria |
| Proposte isolate | sinossi singole non da saga |
