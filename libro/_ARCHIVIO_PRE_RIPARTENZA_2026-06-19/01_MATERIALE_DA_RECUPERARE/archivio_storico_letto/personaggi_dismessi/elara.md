# ELARA

Documento legacy.

Confine di uso:
- questo file e' legacy e non canonico
- serve solo a ricordare cosa e' stato assorbito nel ramo attivo

La linea `Elara` non e' piu' un file canonico attivo.
Le funzioni narrative forti di questo ramo sono state assorbite soprattutto in:
- `personaggi/reva.md`
- `personaggi/secondari.md`
- `riferimenti/voci_narrative.md`

Da questa linea restano utili solo alcuni principi:
- la Lettrice non deve essere addolcita
- il costo percettivo della Lettura deve restare centrale
- il finale di soglia deve chiedere un prezzo personale, non solo cosmico

Nota di manutenzione:
- non espandere questo file
- se un principio qui resta utile, va riversato nei file vivi e basta
