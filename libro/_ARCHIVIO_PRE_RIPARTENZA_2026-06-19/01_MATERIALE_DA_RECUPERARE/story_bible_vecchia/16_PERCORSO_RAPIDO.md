# 16 - PERCORSO RAPIDO

Confine di uso:
- questo file serve solo a orientare il lavoro veloce
- non decide contenuti
- non sostituisce i file madre

Se devo decidere una regola del racconto:
- `00_DECISIONI_APPROVATE.md`
- `03_struttura.md`

Se devo ricordare il cuore della saga:
- `00_scheletro_centrale.md`
- `01_concept.md`

Se devo trovare un termine o una meccanica:
- `04_glossario.md`
- `02_magia.md`

Se devo capire come funziona un libro intero:
- `05_piano_espansione.md`
- `11_AUDIT_LIBRI_E_BUCHI_FUNZIONALI.md`

Se devo chiudere una verita' grossa della saga:
- `06_PROGRAMMAZIONE_GLOBALE.md`

Se devo controllare la storia lunga:
- `07_TIMELINE_E_RIVELAZIONE.md`
- `09_TIMELINE_PERSONAGGI_E_ISTITUZIONI.md`

Se devo controllare mistero, segreti e rivelazioni:
- `08_MATRICE_RIVELAZIONI.md`
- `10_SORGENTI_DI_VERITA_E_ESPOSIZIONE.md`
- `12_MAPPA_SEGRETI.md`

Se devo capire dove scrivere una correzione:
- `13_INDICE_CANONE_E_ANTIRIDONDANZA.md`

Se devo lavorare sul Libro 1:
- `14_AUDIT_BOZZE_LIBRO1.md` per il quadro intero
- `15_MAPPA_CAPITOLI_LIBRO1.md` per il capitolo preciso

Se non devo aprire roba inutile:
- non partire dai file legacy dei vecchi protagonisti
- non partire da `_archivio/` (le vecchie decisioni proposte sono finite li')
- non partire dalle bozze se prima non e' chiara la funzione da verificare

Nota di manutenzione:
- se cambia la gerarchia d'uso dei file guida, si aggiorna qui e poi `00_README.md`
